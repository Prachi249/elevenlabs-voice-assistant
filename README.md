A Python-based virtual voice assistant using **ElevenLabs Conversational AI**.  
The assistant listens through your microphone, transcribes speech, and responds with spoken audio in real time.

---

## Features
- Real-time voice interaction
- Microphone input + speaker output
- Agent-based conversational AI (ElevenLabs)
- Custom callbacks for user speech and agent responses

---

## Requirements
- Python **3.10+**
- macOS (tested on macOS)
- Microphone access enabled
- ElevenLabs account with:
  - An Agent created
  - API key with **ElevenLabs Agents ‚Üí Write** permission
